#### 주의사항: SSG 제약

`output: 'export'` 모드에서는 아래 Next.js 기능을 사용할 수 없다:
- Server Components (서버 사이드 렌더링 부분)
- API Routes (`/api/*`)
- Middleware
- Image Optimization (Next.js 내장)
- ISR (Incremental Static Regeneration)

계산기 사이트는 모든 계산을 클라이언트 JS로 처리하고, 페이지는 빌드 타임에 정적 생성하므로 위 제약이 문제되지 않는다. 이미지는 `unoptimized: true`로 설정하고 Cloudflare의 CDN 캐싱에 의존한다.

### 5. 12주 로드맵 현실성

| 주차 | 계획 | 주말 4~5시간 내 가능 여부 | 판정 |
|---|---|---|---|
| W1 | 도메인 + 스캐폴딩 + Cloudflare 연결 | 3~4시간 예상. 계정 생성에 시간 소요 가능 | 통과 |
| W2 | 연봉 실수령액 계산기 1종 + 가이드 | 4대보험·소득세 로직이 복잡. 5시간 빠듯할 수 있음 | 주의 |
| W3 | 퇴직금 + 주휴수당 2종 | 로직 비교적 단순. 4~5시간 내 가능 | 통과 |
| W4 | 중고차 취득세 + 주담대 2종 | 주담대 상환 스케줄 계산이 중간 난이도. 가능 | 통과 |
| W5 | 전세대출 + 종소세 + 필수 페이지 | 종소세 누진세율이 복잡. 5시간 빠듯 | 주의 |
| W6 | AdSense 신청 + 최적화 | 신청 자체는 30분. 최적화에 시간 투자 | 통과 |
| W7~12 | 주 2종 + 숏폼 | W8부터 숏폼 병행 시 시간 분배 필요 | 주의 |

#### W2, W5 대응책

연봉 실수령액(W2)과 종합소득세(W5)는 로직이 복잡하므로, 해당 주에는 1종에 집중하고 나머지를 다음 주로 이월하는 유연성을 둔다. 로드맵은 목표이지 엄격한 마감이 아니다.

#### 주의: 자연 트래픽

AdSense 승인 시 트래픽이 전혀 없으면 거절 가능성 있음. Search Console에 sitemap 조기 제출(W1~W2)하고, 네이버 서치어드바이저에도 등록해 인덱싱을 앞당긴다.

### 7. SEO 전략 현실성

| 항목 | 검증 결과 | 상태 |
|---|---|---|
| SSG의 SEO 이점 | 정적 HTML → 크롤러 친화적, 로딩 빠름 | 통과 |

| 경쟁 상황 | "연봉 실수령액 계산기" 등은 경쟁 치열하나 기존 사이트 UX 낡음 | 도전적이나 가능 |
| 6개월 10만 PV 목표 | 공격적 목표. 3~5만 PV가 현실적일 수 있음 | 주의 |

#### 주의: PV 목표 조정

6개월 10만 PV는 낙관적 시나리오다. 보수적으로는 3~5만 PV, 낙관적으로는 10만 PV. AdSense 수익도 보수적 시나리오에서 월 5~10만원, 낙관적 시나리오에서 15~30만원으로 범위를 두는 것이 현실적이다.
